# Netflixify App
This video streaming application enables users to browse a categorized film library, perform instant searches, stream movies with adaptive quality based on internet speed, manage favorites, track view counts, and authenticate via Google OAuth.

---

## The Application
The Netflixify App delivers a premium streaming experience with the following features:

- **Categorized Library**: Browse a well-organized collection of films by category.
- **Instant Search**: Quickly find movies with live search functionality.
- **Adaptive Streaming**: Stream videos with multiple quality options, adjusting dynamically to internet speed.
- **Manual Quality Control**: Choose your preferred video quality manually.
- **Favoritesaré List**: Save movies to a personal favorites list for easy access.
- **View Tracking**: Monitor view counts for each film.
- **Secure Login**: Authenticate effortlessly using Google OAuth.
- **User Experience**: Enjoy a responsive, intuitive interface designed for all devices.

---

## The Control Panel
The Netflixify Control Panel empowers administrators with robust tools for efficient app management:

- **Secure Access**: Restricted login for authorized administrators.
- **Dashboard**: Oversee app stats, including view counts, user activity, and movie performance.
- **Category Management**: Add, edit, delete, search, and export movie categories.
- **Movie Management**:
  - Upload movies with detailed metadata.
  - Encode films into multiple quality levels for adaptive playback.
  - Track encoding progress in real-time.
  - Edit, delete, publish, or unpublish movies as needed.
- **Role Management**: Define, edit, and assign roles and permissions for admins.
- **Admin Oversight**: Add, edit, delete, search, and export admin profiles.
- **Customer Management**: Fully manage user accounts with add, edit, delete, search, and export options.
- **App Settings**: Customize OAuth settings (Google, Facebook) and other configurations.
- **Profile Management**: Admin Can Edit Info and change password.

---

## Tools and Technologies
Netflixify is built using cutting-edge tools for top-tier performance:

- **Front-End**: HTML, CSS, and Bootstrap for a responsive design; Livewire for dynamic, real-time interfaces.
- **Back-End**: PHP Laravel for secure, scalable server-side logic.
- **Database**: MySQL for efficient data storage and retrieval.
- **Authentication**: Google OAuth for secure and seamless login.
- **Video Processing**: Advanced encoding tools for multi-quality streaming [**_ffmpeg_**].

---

## Future Features
We’re dedicated to evolving Netflixify with:

- Multi-language support for a global audience.
- AI-driven movie recommendations based on user preferences.
- Integration with additional social login options (e.g., Facebook, Twitter).
- User can Leave comment, ratings and reviews.

Netflixify App blends a seamless streaming experience with powerful admin tools, making it the ultimate choice for modern video entertainment.

# تطبيق Netflixify
تطبيق بث الفيديو هذا يتيح للمستخدمين تصفح مكتبة أفلام مصنفة، إجراء عمليات بحث فورية، بث الأفلام بجودة متكيفة بناءً على سرعة الإنترنت، إدارة المفضلات، تتبع عدد المشاهدات، والمصادقة عبر Google OAuth.

---

## التطبيق
يوفر تطبيق Netflixify تجربة بث متميزة مع الميزات التالية:

- **مكتبة مصنفة**: تصفح مجموعة منظمة من الأفلام حسب الفئة.
- **بحث فوري**: ابحث عن الأفلام بسرعة باستخدام وظيفة البحث المباشر.
- **بث متكيف**: قم ببث الفيديوهات بخيارات جودة متعددة، تتكيف ديناميكيًا مع سرعة الإنترنت.
- **التحكم اليدوي بالجودة**: اختر جودة الفيديو المفضلة يدويًا.
- **قائمة المفضلة**: احفظ الأفلام في قائمة مفضلة شخصية للوصول السريع.
- **تتبع المشاهدات**: راقب عدد المشاهدات لكل فيلم.
- **تسجيل دخول آمن**: قم بالمصادقة بسهولة باستخدام Google OAuth.
- **تجربة المستخدم**: استمتع بواجهة متجاوبة وبديهية مصممة لجميع الأجهزة.

---

## لوحة التحكم
تمكن لوحة تحكم Netflixify المسؤولين من استخدام أدوات قوية لإدارة التطبيق بكفاءة:

- **الوصول الآمن**: تسجيل دخول مقيد للمسؤولين المصرح لهم.
- **لوحة التحكم**: تابع إحصائيات التطبيق، بما في ذلك عدد المشاهدات، نشاط المستخدمين، وأداء الأفلام.
- **إدارة الفئات**: أضف، عدّل، احذف، ابحث، وصدر فئات الأفلام.
- **إدارة الأفلام**:
  - رفع الأفلام مع بيانات وصفية مفصلة.
  - ترميز الأفلام إلى مستويات جودة متعددة للبث المتكيف.
  - تتبع تقدم الترميز في الوقت الفعلي.
  - تعديل، حذف، نشر، أو إلغاء نشر الأفلام حسب الحاجة.
- **إدارة الأدوار**: حدد، عدّل، وخصص الأدوار والصلاحيات للمسؤولين.
- **إدارة المسؤولين**: أضف، عدّل، احذف، ابحث، وصدر ملفات المسؤولين.
- **إدارة العملاء**: إدارة حسابات المستخدمين بالكامل مع خيارات الإضافة، التعديل، الحذف، البحث، والتصدير.
- **إعدادات التطبيق**: تخصيص إعدادات OAuth (Google، Facebook) والتكوينات الأخرى.
- **إدارة الملف الشخصي**: يمكن للمسؤول تعديل المعلومات وتغيير كلمة المرور.

## الأدوات والتقنيات
تم بناء Netflixify باستخدام أدوات متطورة لتحقيق أداء عالي الجودة:

- **واجهة المستخدم**: HTML، CSS، وBootstrap لتصميم متجاوب؛ Livewire لواجهات ديناميكية فورية.
- **الخلفية**: PHP Laravel لمنطق خادم آمن وقابل للتطوير.
- **قاعدة البيانات**: MySQL لتخزين واسترجاع البيانات بكفاءة.
- **المصادقة**: Google OAuth لتسجيل دخول آمن وسلس.
- **معالجة الفيديو**: أدوات ترميز متقدمة لبث متعدد الجودة [**_ffmpeg_**].
